# 🏛️ Comparativa de Versiones: Aquarine

Aquí tienes el detalle técnico de los dos pilares del proyecto para entender cómo evolucionó la herramienta.

## 1. Aquarine Legacy Beta Edition 🕰️
* **Enfoque:** El código base original de cuando inicié el proyecto.
* **Estado:** Edición histórica / Parches menores de estabilidad.
* **Propósito:** Funciona como el cimiento de la arquitectura de Aquarine y registro de mis primeros pasos en GitHub.

## 2. Aquarine Versión Normal (Actual) 🦠
* **Enfoque:** Desarrollo activo enfocado en malware educativo y pruebas de concepto para Android.
* **Plataforma:** Desarrollado visualmente en Sketchware (compila a Java nativo).
* **Propósito:** Pruebas controladas de payloads, bloqueadores visuales y manipulación de interfaces en entornos seguros (VMs).
